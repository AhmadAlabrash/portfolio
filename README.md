## ⚙️ Installation & Setup

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/ahmadalabrash/portfolio.git

# Navigate into the project folder
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```
